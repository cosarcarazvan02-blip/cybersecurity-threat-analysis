# Cybersecurity Threat Detection

ML project that classifies network traffic as normal or malicious using Random Forest, SMOTE, and GridSearchCV.

## Technologies
Python, scikit-learn, imbalanced-learn, pandas, Google Colab

## How to Run
1. Open `Proiect-2.ipynb` in Google Colab
2. Upload `cybersecurity_threat_dataset.csv`
3. Run all cells
